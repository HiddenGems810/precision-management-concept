# Precision Management — Export Directory

This folder structure is the designated save location for all PDF templates exported from the Template Production System.

## Organization
- `/property-flyers`
- `/leasing-brochures`
- `/floor-plans`
- `/move-in-kits`
- `/lease-renewals`
- `/resident-notices`
- `/business-cards`
- `/staff-operations`

## Print Specifications
When printing these PDFs:
- Select **100% Scale** (Do not use "Fit to Page")
- Enable **Background Graphics**
- Disable **Headers and Footers**
- Use high-quality or heavy-stock paper for resident-facing materials (brochures, flyers)
